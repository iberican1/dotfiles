Keybindings	
W : WINDOW	
C : CONTROL	
S : SHIFT	
A : ALT	
Applications	
Keybind	Action
W + Return	Open terminal (alacritty)
W + S + Return	Open fullscreen terminal
W + f	Open file manager (thunar)
W + e	Open text editor (geany)
W + w	Open web browser (firefox)
Menu and Applets	
Keybind	Action
W + d	Open application launcher
W + r	Open command runner
W + n	Open network manager
W + b	Open bluetooth applet
W + m	Open media player
W + x	Open powermenu
W + s	Open screenshot menu
Window Management	
Keybind	Action
W + c/q	Close application
W + MB_LEFT	Move windows
W + MB_RIGHT	Resize windows
W + SCROLL	Zoom in/out
W + A + SCROLL	Change opacity
W + C + MB_RIGHT	Rotate windows
W + C + f	Fisheye effect
W + z	Toggle idle
W + Space	Toggle fullscreen
W + S + t	Toggle always on top
W + S + s	Toggle sticky
W + Left/Right/Up/Down	Grid : Position the windows in certain regions of the output
W + C + Left/Right/Up/Down	Vswitch : Switch to workspace
W + C + S + Left/Right/Up/Down	Vswitch : Move the focused window to respective workspace
A + Tab	Change active window with an animation
A + S + Tab	Change active window (reverse) with an animation
W + Tab	Simple active window switcher
C + A + MB_LEFT	Show the current workspace row as a cube
C + A + Left/Right	Switch to the next or previous workspace (cube)
Window/Super	Expo : Show an overview of all workspaces
Window/Super, 1..9	Expo : Select a workspace by pressing the number
W + o	Switch to the next output
W + S + o	Move window to the next output
W + i	Invert the colors of the whole output
W + v	Shows all the windows on the current workspace
W + S + v	Shows all the windows on all workspaces
Misc Keys	
Keybind	Action
W + p	Color Picker
C + A + l	Lockscreen
Hardware Keys	
Keybind	Action
Print	Take screenshot of desktop
A + Print	Take screenshot of desktop in 5 seconds
S + Print	Take screenshot of desktop in 10 seconds
W + Print	Take screenshot of selected area
XF86AudioRaiseVolume	Increase Volume
XF86AudioLowerVolume	Decrease Volume
XF86AudioMute	Toggle Mute Speakers
XF86AudioMicMute	Toggle Mute Microphone
XF86MonBrightnessUp	Increase Screen Brightness
XF86MonBrightnessDown	Decrease Screen Brightness
